## ✨ Features

- 📊 **Stock Price Prediction**: Predicts future stock prices using historical data.
- 🤖 **AI Chatbot**: Chatbot answers questions about predicted stock data.
- 🧰 **Interactive UI**: Clean, web-based interface built with Streamlit.
